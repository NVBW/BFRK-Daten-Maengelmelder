# BFRK-Daten-Maengelmelder

Jede Person kann gefundene Mängel zum gesamten Datensatz oder zu einzelnen Daten melden, indem Sie einen Issue erstellen und dort möglichst detailliert beschreiben, welchen Fehler Sie melden möchten.

Eine Meldung zu einem konkreten einzelnen Datenfehler sollte mindestens bei der Beschreibung des Fehlers die betroffene Datei und den Name des Datensatzes (Feld ID) beinhalten.
